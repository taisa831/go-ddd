## 集約の基本的構造
集約には境界とルートが存在します

- 境界
	- 集約の境界は集約に何が含まれるのかを定義するための境界です。
- ルート
	- 集約のルートは集約に含まれる特定のオブジェクトです。


ユーザ名の変更はUserオブジェクトに依頼する

```go
userName := NewUserName("NewName")

// NG
user.Name = userName

// OK
user.ChangeName(userName)
```
